# Developer Portfolio

## Português (BR)

### Descrição

Este projeto é uma página de portfólio de desenvolvedor construída com React, Vite e Tailwind CSS. Ele inclui animações de tecnologia e componentes visuais modernos.

### Principais Tecnologias

- React
- Vite
- Tailwind CSS
- Framer Motion
- SVGs para logos de tecnologia

### Como executar localmente

1. Instale as dependências:

```bash
npm install
```

2. Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

3. Abra o endereço exibido no terminal (normalmente `http://localhost:5173`).

### Como gerar a versão de produção

Execute:

```bash
npm run build
```

O conteúdo final será gerado na pasta `dist/`.

### Como pré-visualizar a build de produção

Execute:

```bash
npm run preview
```

### Observações

- Mantenha as logos SVG na pasta `src/assets` para facilitar os imports no aplicativo.
- Ajuste classes Tailwind em `src/App.jsx` conforme necessário para personalizar o layout.

## English

### Description

This project is a developer portfolio landing page built with React, Vite, and Tailwind CSS. It includes animated technology stack cards and modern visual styling.

### Key Technologies

- React
- Vite
- Tailwind CSS
- Framer Motion
- SVG logos for technology icons

### Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the URL shown in the terminal (usually `http://localhost:5173`).

### Build for production

Run:

```bash
npm run build
```

The production-ready files will be created in the `dist/` folder.

### Preview the production build

Run:

```bash
npm run preview
```

### Notes

- Keep SVG logo assets in `src/assets` for easy imports.
- Edit Tailwind classes in `src/App.jsx` to customize the layout and styling.
